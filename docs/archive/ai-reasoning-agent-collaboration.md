## The Collaborative Reasoning Process

### Phase 1: Information Gathering & Validation
Data agents collect and cross-validate information from multiple sources, debating data quality and reaching consensus on market signals.

### Phase 2: Proposal Generation
Strategy agents independently generate investment proposals using their specialized approaches.

### Phase 3: Debate & Challenge
Agents engage in structured debate:
- Strategy agents challenge each other's assumptions
- Risk agent provides counterarguments and risk perspectives
- Data agents validate underlying assumptions
- Reflection agent ensures logical consistency

### Phase 4: Risk Assessment & Refinement
Risk agent performs comprehensive probabilistic analysis, with other agents providing input on risk factors and mitigation strategies.

### Phase 5: Consensus Building
Through iterative deliberation, agents reach consensus on optimal strategies, with reflection agent mediating conflicts.

### Phase 6: Execution Validation
Execution agent validates practical feasibility, with final sanity checks from all agents.

### Phase 7: Learning Integration
Learning agent incorporates outcomes into future reasoning processes, continuously improving the collective intelligence.
